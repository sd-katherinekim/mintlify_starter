## Global customization

[](https://sendbird.com/docs/chat/uikit/v3/android-view/customizations/overview-customizations#2-global-customization)

Feedback

You can apply customizations to the entire UIKit framework by utilizing the `FragmentProviders`, `ModuleProviders`, `AdapterProviders`, and `ViewModelProviders` classes provided by UIKit.​

* [Customize fragments](https://sendbird.com/docs/chat/uikit/v3/android-view/customizations/global-customization/fragments)

* [Customize modules and components](https://sendbird.com/docs/chat/uikit/v3/android-view/customizations/global-customization/modules-and-components)

* [Customize ViewModels](https://sendbird.com/docs/chat/uikit/v3/android-view/customizations/global-customization/viewmodels)

* [Customize list adapters](https://sendbird.com/docs/chat/uikit/v3/android-view/customizations/global-customization/list-adapters)

## Resource customization

[](https://sendbird.com/docs/chat/uikit/v3/android-view/customizations/overview-customizations#2-resource-customization)
